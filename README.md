# osCommerce  歐付寶金流
---

## 1. 介紹

  - 歐付寶金流外掛套件(以下簡稱外掛套件)，提供合作特店以及個人會員使用開放原始碼商店系統時，無須自行處理複雜的檢核，直接透過安裝設定外掛套件，便可以較快速的方式介接金流系統 
  - 收款方式清單：
    - 信用卡(一次付清、分期付款、定期定額)
    - 網路ATM
    - ATM櫃員機
    - 超商代碼
    - 歐付寶餘額
    
 

## 2. 適用版本
  - osCommerce Version 2.3.4
